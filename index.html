Welcome to the Layout Generator.

You can choose your own template, color theme, and if you want to apply a set background image (unique to each color theme and for each template). Then we will generate the HTML, CSS, and JavaScript code for you to use! We are assuming that you already have a basic understanding of web page creation. Let's get started!
